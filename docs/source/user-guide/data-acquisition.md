# Data Acquisition (Removed)

```{note}
Hardware data acquisition via VNA/myVNA has been removed from RheoQCM.
This package now focuses on analysis of previously collected data.
```

For importing QCM data collected with other instruments, see {doc}`data-import`.

## Supported Import Formats

- **QCM-D** format (`.xlsx`) via `File > Import QCM-D`
- **QCM-Z** impedance format via `File > Import QCM-Z`
- **HDF5** files (`.h5`) via `File > Load Exp`
